# クラス応用

## 抽象クラス
```py
import abc

class Person(metaclass=abc.ABCMeta):

    @abc.abstractmethod
    def drive(self):
        pass
``` 
pythonにはもともと抽象クラスがなかったが、Java等と比較したときに必要となったため、上記のように利用している。

## 多重継承
```py
class PersonCarRobot(Person, Car):
    def fly(self):
        print('fly')
```
上記のように、継承クラスを並べれば多重継承が可能。メソッドは左側に書いたものが優先されるので注意。

## クラス変数
```py
class T(object):
    words = []
    def add_word(self, word):
        self.words.append(word)

c = T()
c.add_word('add 1')
c.add_word('add 2')

d = T()
c.add_word('add 1')
c.add_word('add 2')

print(c.words)
# ['add 1', 'add 2', 'add 1', 'add 2']
```
クラス変数はオブジェクトで共有して利用されるため、変更がないものを入れるのが基本。

## クラスメソッド
```py
class Person(object):
    kind = 'human'

    @classmethod
    def what_is_your_kind(cls):
        return cls.kind

# オブジェクトを作る
a = Person()
print(a.what_is_your_kind())

# クラスを指し示す
b = Person
print(b.what_is_your_kind())
```
オブジェクトを生成しない状態でも、クラスメソッドを使えばアクセスできるようになる。

## スタティックメソッド
```py
class Person(object):
    kind = 'human'
    @staticmethod
    def about(year):
        print(f'about human {year}')

Person.about(1999)
# about human 1999
```
