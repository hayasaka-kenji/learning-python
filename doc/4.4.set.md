# データ構造

## 集合型
```py
> a = {1, 2, 3, 4, 4, 5, 5, 6 }  # {1, 2, 3, 4, 5, 6}
> type(a)
<class 'set'>
 b = {2, 3, 3, 6, 7}  # {2, 3, 6, 7}
```
`{}`で囲んでリストのように定義すると集合型を作れる。

```py
# aからbの値を取り除く
> a - b  
{1, 4, 5}
> b - a
{7}

# aかつb
> a & b
{2, 3, 6}

# aにもbにもある
> a | b
{1, 2, 3, 4, 5, 6, 7}

# どちらかにあるが重複していないもの
> a ^ b
{1, 4, 5, 7}
```
- `a + b` はエラーになるので注意。
- 集合にはインデックスはない。

### メソッド
```py
> s = {1, 2, 3, 4, 5}
> s.add(0)  # {0, 1, 2, 3, 4, 5}
> s.remove(0)  # {1, 2, 3, 4, 5}
> s.clear() # set()
```
- 集合の方は`set()`となる。
