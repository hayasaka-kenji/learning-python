# while文
```py
count = 0
while count < 5:
    print(count)
    count += 1
```
`count` を書き忘れると無限ループになるので注意。

## break, continue
```py
count = 0
while True:
    if count >= 5:
        break

    if count == 2:
        count += 1
        continue

    print(count)
    count += 1
```
- `break` プログラムを抜ける。
- `continue` スキップする。

## else
```py
count = 0

while count < 5:
    print(count)
    count += 1
else:
    print('done')
```
- `while`の式がfalseのとき、else内の命令が実行される。
- `break`が入っていると、抜けてしまうので注意。
- while-elseは使用頻度が低いが、開発のときに使うことがある。

## input関数
```py
while True:
    word = input('Enter? ')
    if word == 'ok':
        print('Success!')
        break
    print('Next')
```
whileと併せてよく使われる。
- 受け付けた値はstr型なので注意が必要。
