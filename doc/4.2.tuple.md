# データ構造

## タプル型
```py
t = (1, 2, 3, 4)
> type(t) # <class 'tuple'>
> t[1] = 100 # TypeError: 'tuple' object does not support item assignment

> t = ([1, 2, 4], [2, 4, 6])
> t[0][0] = 100 # ([100, 2, 4], [2, 4, 6])

> t = 1,
> type(t) # <class 'tuple'>
```

- 丸括弧を使って囲む
- リストに似ているが再代入できない構造。新しい値をアサインすることをサポートしていない。
- メソッドも少ない。(`index`や`count`はできる。))
- 中にリストを入れて、リスト内の修正をすることは可能。
- 括弧がなくてもカンマ区切りで定義可能。1つでもカンマをつければタブル型になる。
- 新しいタプル型を作れば再代入のようなことはできる。

### アンパッキング
```py
num_tuple = 10, 20
print(num_tuple)

# x, y = 10, 20 と同義
x, y = num_tuple
print(x, y)  # 10 20
```
- 括弧がなくても定義できる。python内ではタプルを使って変数展開しているという考え方になる。

```py
min, max = 0, 100
print(min, max)  # 0 100
a, b, c, d, e, f = 'hoge', 'fuga', 'piyo', '1', '2', '3'
```
- 数が多いと、わかりづらくなるので注意。

### 変数の入れ替え
```py
i = 10
j = 20
tmp = i
i = j
j = tmp
print(i, j)  # 20 10
```
- 3行も使う。

```py
a = 100
b = 200
a, b = b, a
print(a, b)
```
- アンパッキングを利用するとシンプル入れ替えることができる。

### リストとの使い分け
タプルは再定義できないので定数のような使い方を使って、バグ対策に使う。
