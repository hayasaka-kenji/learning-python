# for文
```py
some_list = [1,2,3,4,5]
for i in some_list:
    print(i)
```
- イテレーターを使った繰り返しの際に使う。
- whileより短くシンプルに書くことができる。

```py
# while文の場合
i = 0
while i < len(some_list):
    print(some_list[i])
    i += 1
```

## break, continue
```py
for w in ['my', 'name', 'is', 'mike']:
    if w == 'name':
        continue
    print(w)
# my
# is
# mike
```

## else
```py
for f in ['apple', 'banana', 'orange']:
    print(f)
else:
    print('I age all!')
```
while同様`break`がなければ、終了時にelseが実行される。

## よく使われるメソッド

### range
```py
# 0 - 9が出力される
for i in range(10):
    print(i)

# 2 5 8が出力される
for i in range(2, 10, 3):
    print(i)
```
- 引数を2つ以上にした場合
    - 1つめ引数はいくつからスタートするか
    - 2つめは、繰り返す回数
    - 3つめは、何個飛ばしで実行するか

```py
for _ in range(10):
    print('hello')
```
for関数内で使わない場合`_`と明示的に使わないことを示すことができる。

### enumerate
```py
for i, fruit in enumerate(['apple', 'orange', 'banana']):
    print(i, fruit)
```
インデックスを引き渡すことができる関数。  

### zip
```py
days = ['Mon', 'Tue', 'Wed']
fruits = ['apple', 'banana', 'orange']
drinks = ['coffee', 'tea', 'soda']
for day, fruit, drink in zip(days, fruits, drinks):
    print(day, fruit, drink)
# Mon apple coffee
# Tue banana tea
# Wed orange soda
```
まとめて引数に渡すことができる関数。

## ディクショナリー型をfor文で処理する
```py
d = {'x': 100, 'y': '200'}

for k, v in d.items():
    print(k, ' =>', v)
# x  => 100 , y  => 200

# items()を確認する
print(d.items())
# dict_items([('x', 100), ('y', '200')])
```
- ディクショナリー型をメソッドを使わず回そうとするとkeyのみが出力されてしまう。
- keyとvalueどちらも出力したい場合は、`items()`を使う。
- 内部の仕組みはitemsはディクショナリー型をタプルを変換してくれうようになり、それをアンパッキングする形になる。
