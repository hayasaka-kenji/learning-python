# if文
```py
x = 10

if x < 0:
    print('negative')
elif x == 10:
    print('10')
elif x == 0:
    print('zero')
else:
    print('positive')
```
- `if else elif` でif文を書く。
- 括弧を使わずにインデントを使う。
- インデントがずれているとエラーになる。

## 論理演算子
```py
a, b = 1, 1

# かつ
print(a > 0 and b > 0)

# または
print(a > 0 or b > 0)
```
- or はif文で書くと長くなる。

## in, not
```py
y = [1, 2, 4]
x = 1

if x in y:
    print('in')

if 100 not in y:
    print('not in')
```
inはAの中にBが入っているか確認するときに使う。

```py
a = 1
b = 2

# if not a == b:
#     print('not equal')

# 以下のように書き換えたほうがわかりやすい
if a != b:
    print('not equal')

is_ok = True

# == Trueとしなくていい
if is_ok:
    print('ok')

# 出力されない
if not is_ok:
    print('ok')
```
notは否定する際に使う。`!=`などのほうがわかりやすく、シンプルなときも多い。

### None
```py
is_empty = None
# print(help(is_empty))

if is_empty is None:
    print('None!!')

# True
print(1 == True)
# False
print(1 is True)
# True
print(True == True)
```
- 変数は設定するが、なにも入れたくないときに使う。
- `is` はオブジェクト同士に使われる。Noneオブジェクトとの比較のときに使われることが多い。
