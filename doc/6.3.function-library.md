# 組み込み関数
もともとpythonがインタプリタに組み込んでいていつでも使えるようになっている。

```py
import builtins
builtins.print()
```
上記のような表記は省略することができている。

組み込み関数の一覧
https://docs.python.org/ja/3/library/functions.html
