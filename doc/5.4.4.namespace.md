# 名前空間とスコープ

## global
```py
animal = 'cat'

def f():
    # エラー
    print(animal)
    animal = 'dog'
    # こっちはうまくいく
    print(animal)

f()
```
- グローバルの変数は使えない。

```py
# 名前空間
animal = 'cat'

def f():
    global animal
    print(animal)
    animal = 'dog'
    print(animal)

f()
```
- グローバル変数を使うときは、`global`を使う。

```py
def f():
    animal = 'cat'
    print('local:' ,locals())

f()
print('global:', globals())
# local: {'animal': 'cat'}
# global: {'__name__': '__main__', '__doc__': None, ...
```
- `local()`や`global()`を行うと変数適宜されているものが表示される。
- `__name__`のようにアンダースコアを使っているものをpythonで定義されているものなので、書き換える際はよく理解して置くようにする。
