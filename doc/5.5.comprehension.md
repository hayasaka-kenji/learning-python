# 内包表記

## リスト内包表記
```py
t = (1, 2, 3, 4, 5)

# forループを使った書き方
r = []
for i in t:
    if i % 2 == 0:
        r.append(i)
print(r)

# 内包表記を使った書き方。
r = [i for i in t if i % 2 == 0 ]
print(r)
```
ループを短く書くことができる。
(使えるから使うではなく、読みやすいかどうかで判断するようにする。)

## 辞書内包表記
```py
w = ['mon', 'tue', 'wed']
f = ['coffee', 'milk', 'water']
d = {x: y for x, y in zip(w, f)}
print(d)
# {'mon': 'coffee', 'tue': 'milk', 'wed': 'water'}
```

## 集合内包表記
```py
s = set()
s = {i for i in range(10) if i % 2 == 0}
print(s)
# {0, 2, 4, 6, 8}
```

## ジェネレーター内包表記
```py
def g():
    for i in range(10):
        yield i

g = g()
g = (i for i in range(10) if i % 2 == 0)

for x in g:
    print(x)
# 0 2 4 6 8
```
- タプルと似ているので注意。
- タプルで内包表記したい場合は、タプルを宣言するようにする。
